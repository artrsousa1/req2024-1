    Requisitos elicitados

    Esta seção é destinada para a elicitação dos requisitos de todas as áreas definidas da Play Store.

    Requisito Definição Técnica de elicitação Prioridade
    REQ01 O usuário deve poder buscar por um app ou jogo Análise de protocolo e introspecção Must
    REQ02 O usuário deve poder instalar um app ou jogo Análise de protocolo e introspecção Must
    REQ03 O usuário deve poder atualizar um app ou jogo Análise de protocolo e introspecção Must
    REQ04 O usuário deve poder desinstalar um app ou jogo Análise de protocolo e introspecção Must
    REQ05 O sistema deve fornecer a política de privacidade de um app ou jogo Análise de protocolo Must
    REQ06 O sistema deve mostrar em quais sistemas operacionais um app ou jogo pode ser executado Análise de protocolo Must
    REQ07 O sistema deve mostrar qual é a data de lançamento de um app ou jogo Análise de protocolo Must
    REQ08 O sistema deve mostrar quais são os dados que um app ou jogo extrai dos usuários Análise de protocolo Must
    REQ09 O sistema deve mostrar quais são as permissões que um app ou jogo tem na máquina do usuário Análise de protocolo Must
    REQ10 O sistema deve mostrar o que um app ou jogo faz com os dados extraídos dos seus usuários Análise de protocolo Must
    REQ11 O usuário deve poder pedir a deleção dos dados coletados por um jogo ou app Análise de protocolo Must
    REQ12 O usuário deve poder comprar um app ou jogo Observação Must
    REQ13 O usuário deve poder pedir reembolso por um app ou jogo Observação Must
    REQ14 O aplicativo deve permitir o login com uma conta Google já existente. Introspecção, Observação e Storytelling Must
    REQ15 O aplicativo deve autenticar a senha ao fazer login com uma conta. Introspecção e Observação Must
    REQ16 O aplicativo deve permitir a remoção de contas logadas. Introspecção e Observação Must
    REQ17 O aplicativo deve permitir a troca da conta principal utilizada no dispositivo. Introspecção e Observação Must
    REQ18 O aplicativo deve manter salvas as formas de pagamento da conta. Introspecção, Observação e Storytelling Must
    REQ19 O aplicativo deve ser capaz de fornecer a criação de uma conta Gmail (Google Email). Introspecção e Observação Should
    REQ20 O sistema deve separar apps e jogos em categorias Análise de protocolo e introspecção Should
    REQ21 O sistema deve apresentar o espaço de armazenamento de um app ou jogo Análise de protocolo, introspecção Should
    REQ22 O sistema deve apresentar o progresso de downloads Análise de protocolo Should
    REQ23 O sistema deve persistir o histórico de apps ou jogos já instalados por aquele e-mail Introspecção, análise de protocolo Should
    REQ24 O usuário deve poder sinalizar um app ou jogo como inapropriado Introspecção, análise de protocolo Should
    REQ25 O sistema deve ter um tutorial básico para novos usuários Storytelling Should
    REQ26 O sistema deve ser acessível para pessoas idosas Storytelling Should
    REQ27 O usuário deve poder executar um app ou jogo a partir da Play Store Análise de protocolo Should
    REQ28 O sistema deve mostrar quais são as compras possíveis dentro de um app ou jogo, caso existam Storytelling Should
    REQ29 O aplicativo deve permitir o gerenciamento de aplicativos, livros e filmes vinculados ao Gmail. Introspecção e Observação Should
    REQ30 O sistema deve mostrar qual a empresa criadora do app ou jogo Análise de protocolo Should
    REQ31 O aplicativo deve fornecer acesso à funcionalidade "esqueci a senha" do Gmail. Introspecção e Observação Should
    REQ32 O aplicativo deve permitir que o usuário contate o suporte. Introspecção e Observação Should
    REQ33 O aplicativo deve permitir configurações de acessibilidade associadas ao usuário. Introspecção Should
    REQ34 O aplicativo deve permitir configurações de segurança associadas ao usuário. Introspecção Should
    REQ35 O aplicativo deve permitir a mudança de dispositivos de forma fácil, oferecendo backup e reinstalação de aplicativos, filmes e livros de uma conta Gmail. Storytelling Should
    REQ36 O usuário deve poder avaliar um app ou jogo Análise de protocolo e introspecção Could
    REQ37 O sistema deve apresentar a classificação indicativa de apps e jogos Storytelling e introspecção Could
    REQ38 O sistema deve apresentar os apps ou jogos mais bem avaliados pela comunidade Introspecção Could
    REQ39 O sistema deve apresentar os apps ou jogos mais vendidos Introspecção Could
    REQ40 O sistema deve apresentar os apps ou jogos em alta Introspecção Could
    REQ41 O sistema deve apresentar jogos ou apps similares àquele que está em foco pelo usuário Análise de protocolo e introspecção Could
    REQ42 O sistema deve recomendar apps ou jogos de acordo com o perfil de um usuário Introspecção Could
    REQ43 O usuário deve poder criar uma lista de apps e/ou jogos de desejo Introspecção Could
    REQ44 O usuário deve poder escrever uma resenha de um app ou jogo Análise de protocolo Could
    REQ45 O sistema deve fornecer o contato para suporte com os desenvolvedores de app ou jogo Análise de protocolo Could
    REQ46 O sistema deve mostrar quando um app ou jogo foi atualizado pela última vez e qual é sua atual versão Análise de protocolo Could
    REQ47 O usuário deve poder visualizar as avaliações que outros usuários fizeram de um app ou jogo Análise de protocolo Could
    REQ48 O usuário deve poder compartilhar apps ou jogos com outros usuários Análise de protocolo Could
    REQ49 O aplicativo deve permitir o gerenciamento de dispositivos vinculados ao Gmail. Introspecção e Observação Could
    REQ50 O aplicativo deve controlar o histórico de comentários e avaliações feitos pelo Gmail. Introspecção e Observação Could
    REQ51 O aplicativo deve notificar o usuário tanto no próprio app quanto via Gmail. Introspecção e Observação Could
    REQ52 O sistema deve mostrar eventos de um app ou jogo Introspecção, análise de protocolo Would
    REQ53 O sistema deve apresentar as últimas novidades de um app ou jogo Análise de protocolo, introspecção Would
    REQ54 O sistema deve ter uma seção específica para aplicativos voltados ao público infantil Introspecção Would
    REQ55 O sistema deve persistir as pesquisas mais recentes de um usuário Introspecção Would
    REQ56 O sistema deve fornecer um tutorial de como o app ou jogo funciona Análise de protocolo Would
    REQ57 O sistema deve mostrar a quantidade de downloads que um app ou jogo tem Análise de protocolo Would
    REQ58 O aplicativo deve permitir configurações personalizadas para cada conta. Introspecção e Observação Would
    REQ59 O aplicativo deve gerenciar os benefícios e ferramentas disponíveis para a Play Store vinculadas ao Gmail. Introspecção e Observação Would
